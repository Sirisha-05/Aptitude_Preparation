📋 Cheat Sheet — All Formulas & Tricks

🔑 Core Definitions

  • Cost Price (CP) — Price at which item is purchased
  • Selling Price (SP) — Price at which item is sold
  • Marked Price (MP) — Label / listed price
  • Profit — SP > CP
  • Loss — CP > SP
  • Discount — Reduction on Marked Price

📐 Essential Formulas

Profit = SP − CP
Loss = CP − SP
Profit% = (Profit / CP) × 100
Loss% = (Loss / CP) × 100
SP = CP × (100 + P%) / 100
SP = CP × (100 − L%) / 100
CP = SP × 100 / (100 + P%)
CP = SP × 100 / (100 − L%)
Discount = MP − SP
Discount% = (Discount / MP) × 100
SP = MP × (100 − Discount%) / 100

⚡ Tricks & Shortcuts

Successive discount (x%, y%): Net discount = x + y − (xy/100)
Same item sold: one at x% gain, one at x% loss → Always LOSS = x²/100 %
CP of n items = SP of m items → Profit% = (n−m)/m × 100
False weight profit% = Error / (True − Error) × 100
Net % change in revenue = x + y + xy/100 (use −ve for loss)
If MP is x% above CP → MP = CP × (100+x)/100

🔢 Fraction ↔ % Quick Table

  • 1/2 = 50% | 1/3 = 33.33% | 1/4 = 25% | 1/5 = 20%
  • 1/6 = 16.67% | 1/7 = 14.28% | 1/8 = 12.5% | 1/9 = 11.11%
  • 1/10 = 10% | 1/11 = 9.09% | 1/12 = 8.33% | 1/20 = 5%

🧠 Common TCS Patterns

  • Pattern 1: CP of x articles = SP of y articles → profit/loss %
  • Pattern 2: Two successive discounts → effective net discount
  • Pattern 3: Trader marks x% above CP, allows y% discount → find gain/loss
  • Pattern 4: Item sold at gain/loss, find alternate SP for target %
  • Pattern 5: Price↑ so quantity↓ — net revenue change
  • Pattern 6: Mixed batch: part at gain%, part at loss% → overall %
  • Pattern 7: False weights → profit % using formula
  • Pattern 8: MP given with two discounts → find CP

📌 Steps to solve any P&L problem

  • 1. Identify CP, SP, MP, discount given
  • 2. Convert all percentages to ratios if needed
  • 3. Use SP formula or CP formula directly
  • 4. For multi-step: chain the % changes
  • 5. Always check — Profit% and Loss% are on CP
  • 6. Discount% is always on MP

🏃 Exam-day speed hacks

  • Assume CP = 100 for % problems
  • Use multipliers: 20% gain → SP = 1.2 × CP
  • Successive discounts: multiply factors (0.9 × 0.8 = 0.72 → 28% net)
  • CP of n items = SP of m items → just substitute ratio
  • Two same price x% gain & x% loss → net loss = x²/100%
